Examples=============In addition, this section of the documentation provides examples in the form of Jupyter notebooks with more detailed explanation, which can be used in a similar fashion.The example workflow of the MISO2 model is provided as notebooks:.. toctree::   :maxdepth: 2   :caption: Contents:   ../notebooks/MISO2_01_create_config.ipynb   ../notebooks/MISO2_02_run_model.ipynb   ../notebooks/MISO2_03_working_with_results.ipynbRun the notebooks in order to create a configuration file from the inputs, run the MISO2 model, and interact with the outputs.
